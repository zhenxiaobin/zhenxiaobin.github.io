一、新建一个Flutter Package工程

Next

设置工程名字，选择Fluter SDK路径，选择本地存放位置

Finish

二、配置pubspec.yaml文件

  • name：工程名
  • description：工程描述
  • version：版本号
  • author：作者信息 邮箱一下要加上，不然后面检查会不通过.
  • homepage：主页 一般填写的github地址

** 以上信息是必填项，不然检测时会不通过，下面会详细介绍

三、库内容

1、库内容存放在lib目录下
2、特别提醒一定要创建一个与工程名字一样的dart文件

3、把包上传到github上，配置README.md和CHANGELOG.md文件，会影响pub发布后的Overall Score。
红色是低分，绿色是高分，好的代码规范会提高分数，
4、文档注释要用 /// 不要用 /* /

四、上传到pub库

提示：记得在当前包下执行一下命令

1、发布前用命令检查

pub publish --dry-run

验证通过

Package has 0 warnings.

2、发布

pub publish

发布成功

Looks great! Are you ready to upload your package (y/n)? y
Uploading… (4.0s)
Successfully uploaded package.

4、命令行被墙

Proxy failed to establish tunnel (500 Internal Server Error)

基于Shadowsocks配置
a、打开终端

vim ~/.zshrc

在下面添加

alias proxy='export all_proxy=socks5://127.0.0.1:1086'
alias unproxy='unset all_proxy'

**关于127.0.0.1:1086的由来
打开Shadowsocks的偏好设置

继续执行命令

source ~/.zshrc
proxy
curl cip.cc
unproxy //关闭

查看ip是否到了国外

b、打开终端

vim /usr/local/etc/privoxy/config

在最下面添加下面两行

listen-address 0.0.0.0:8118
forward-socks5 / localhost:1086 .

**注意一下forward-socks5这行最后面的空格和. 我也不知道是为什么，1086同上面Shadowsocks的偏好设置

sudo /usr/local/sbin/privoxy /usr/local/etc/privoxy/config
export http_proxy='http://localhost:8118'
export https_proxy='http://localhost:8118'
curl cip.cc
